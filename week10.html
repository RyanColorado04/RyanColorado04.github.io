Weekly Update

I got two more practice problems done today. These forced me to use zip as well as option.

Option is a Monad. Monads cover a “computational aspect”. In this problem, it covered the possible absence of a value. From the question explanation: Proper use of Monads can result in very concise yet elegant and readable code.

Here was my solution:


object Hamming {
	def distance(dnaStrandOne: String, dnaStrandTwo: String): Option[Int] = {
		if (dnaStrandOne.length == dnaStrandTwo.length){
			Option(dnaStrandOne zip dnaStrandTwo count (dnaStrand => dnaStrand._1 != dnaStrand._2))
				} else { None }
		}
}

There is probably a better way to do this, but I was trying to solve it in a "Scala" way. I had to watch some YouTube videos and read some blogs and Stack Overflow to get these problems today. I will continue working on these over the weekend as well as reading and watching lectures.
